Round-robin tournament

Every strategy plays every other — 5% signal noise, games end at random

Home Simulations Game Theory

Which strategy survives when nothing is certain — not signals, not game length?

Each pair of strategies plays one game. Every round has a 0.5% chance of being the last, so the expected game length is 200 rounds but no strategy can count on that. Actions are misperceived 5% of the time. Scores accumulate across all matchups — the winner is whoever earns the most points in total.

How this tournament works

This is modelled after Robert Axelrod's 1980 computer tournament, which found that cooperative strategies — particularly tit-for-tat — outperformed more aggressive ones in repeated games. Here we add two complications Axelrod's original tournament did not have:

  • Signal noise (5%): each player independently misperceives the other's action with 5% probability, which can trigger undeserved punishment spirals.
  • Uncertain game length (0.5%/round end probability): strategies cannot defect in the "last round" because they never know when it is coming. Expected game length is 200 rounds.

Both features tend to hurt strategies that punish harshly (like grim trigger) and reward strategies with some tolerance for perceived defection.
